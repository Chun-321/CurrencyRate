# **App Name**: Currency Compass

## Core Features:

- Bank Data Aggregation: Collect current USD buy rates, fees, and short-term promotions from the five specified banks.
- Comprehensive Ranking: Provide a default-weighted ranking of USD exchange recommendations that considers rates, fees, and promotions.
- Best Rate Ranking: Rank banks solely on the USD buy rate, ignoring fees or promotions.
- Personalized Ranking Adjustment: Allow users to adjust ranking priorities to favor either exchange rates or promotional offers, updating the Comprehensive Ranking accordingly.
- Ranking Explanation: Explain why a particular bank ranks higher in the comprehensive ranking based on the combination of factors (exchange rate, fees, promotions). This feature acts as a tool to interpret the data.

## Style Guidelines:

- Primary color: Light teal (#A7DBD8) to evoke trust and financial responsibility, reflecting the app's focus on currency.
- Background color: Very light gray (#F2F4F3), a desaturated version of the primary color for a clean and neutral backdrop.
- Accent color: Pale yellow (#F0E68C), an analogous color, to highlight important UI elements.
- Body and headline font: 'Inter', a sans-serif font providing a modern, machined look.
- Use clean and simple icons to represent different banks and currencies, ensuring clarity.
- Use a clear, tabular layout to present exchange rates and rankings, facilitating easy comparison.
- Employ subtle transitions and animations when the ranking updates to provide a smooth user experience.
